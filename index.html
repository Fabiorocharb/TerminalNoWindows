Conhecendo o terminal no windows

O Terminal no Windows é uma ferramenta imprescindível na vida dos profissionais da TI, do programador ao profissional de redes, pois é uma ferramenta capaz de fazer instalações, configurações, checar dados e navegar pelo sistema.

Conhecendo o CMD

Para abrir o CMD aperte o botão "windows+r" irá abrir o executar, digite CMD e apos "Enter".

Vai aparecer as informações do windows, disco local e nome do usuário.

Funções basicas

"dir"= Serve para mostrar as pastas e arquivos que estão na pasta apontada pelo terminal no Windows.
"cd" = Serve como um duplo clique no mouse.
"cd.." = Serve para voltar.
"cd + nome da pasta" = Serve para avançar para dentro de uma pasta.
"Cls ou clear" = Serve para limpar todos os comandos listados,Lembrando que ele limpa apenas os comandos, nenhum efeito de comando anterior é perdido.
"Mkdir" = Cria uma nova pasta. ex:mkdir + 'NomeDaPasta'
"Del + nome da pasta/Arquivo" = Para apagar ex:del + 'NomeDaPasta/Arquivo'.
"Help" = Um outro comando super válido de se utilizar é o 'help', pois ele lista todos os comandos que você pode utilizar.

Outros usos

Várias tecnologias quando instaladas na sua máquina têm o potencial de serem utilizadas pelo terminal. Um exemplo disso é o NodeJS, uma vez instalada na sua máquina você pode utilizar comandos com o ‘npm’ ou ‘node’, que indicam que você estará utilizando aquele software.
